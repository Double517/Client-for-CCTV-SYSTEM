##应用简介  
基于React native的视频监控App，借助Bilibili的开源项目ijkplayer实现实时直播流视频的播放功能，App接口使用淘宝的rap统一进行管理，目前仅能运行在安卓平台上。  
##功能简介  
<ul>
    <li>直播流播放</li>
    <li>控制小车运动</li>
    <li>摄像头设备管理功能</li>
</ul>
